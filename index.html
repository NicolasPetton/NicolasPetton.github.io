I'm Nicolas Petton

I'm a Smalltalk developer, Emacs lisper, and web enthusiast.

Work

I'm an engineer in the RMoD team at Inria and the co-founder of Objectfusion, a company dedicated into building custom web applications.

Projects

  • I'm the creator of Amber, a Smalltalk that runs inside the web browser
  • I'm also the author of SmalltalkHub, a code repository and project management application for Smalltalk
  • I contribute to MongoTalk, the Pharo driver for MongoDB, and Phriak, the Pharo driver for Riak.

You can learn more about Amber on the GitHub page, and can find more projects on my Github and SmalltalkHub accounts.
